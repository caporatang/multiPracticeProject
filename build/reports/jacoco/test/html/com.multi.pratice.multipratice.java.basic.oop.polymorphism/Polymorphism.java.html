    // ------------------------------ 그래서 상속은 언제 사용해야 좋은건데? ----------------------------
    // IS - A(inheritance)관계
    // 일반적인 개념과 구체적인 개념과의 관계
    // 상위 클래스 : 하위 클래스보다 일반적인 개념
